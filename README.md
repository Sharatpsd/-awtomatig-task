# Task Manager Application

A simple Full Stack Task Manager application built with React and Django REST Framework.

## Overview

This project allows users to manage tasks efficiently through a clean and responsive interface.

Users can:

- Add new tasks
- View all tasks
- Update task status
- Delete tasks

The frontend is built using React and Axios, while the backend is powered by Django REST Framework and SQLite.

---

## Features

### Task Management

- Create tasks with title, description, and status
- View all existing tasks
- Update task status
- Delete tasks instantly

### Status Options

- To Do
- In Progress
- Done

### CRUD Operations

- Create
- Read
- Update
- Delete
